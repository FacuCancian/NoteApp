# 📝 NoteApp — Aplicación de Notas en Android

NoteApp es una aplicación moderna donde podés **agregar notas, editarlas, eliminarlas y compartirlas** con quien quieras.  
Construida con **Jetpack Compose**, **Clean Architecture**, **MVVM**, **Room** y **Dependency Injection**, ofrece una experiencia rápida, intuitiva y con buen diseño.

---

## 🚀 Características principales

- ✏️ Crear notas  
- ✍️ Editar notas existentes  
- 🗑️ Eliminar notas  
- 📤 Compartir notas con cualquier app
- ⏰ Alarma / recordatorio para las notas
- 📝 Búsqueda de notas  
- 🎨 UI moderna con Jetpack Compose  
- 💾 Guardado local con Room  
- 🧩 Arquitectura limpia y escalable  
- ⚡ Fluidez y mejores prácticas de Android moderno  

---

## 🛠️ Tecnologías utilizadas

- **Kotlin**
- **Jetpack Compose**
- **MVVM**
- **Clean Architecture**
- **Room Database**
- **Coroutines & Flow**
- **Hilt / DI**
- **Material Design 3**
